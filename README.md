EPS-Frontend-Structured
========================

This is a static UI mockup for the Exams Practice System (EPS). Launch `index.html` to open the Login/Sign-up page.

Files:
- `index.html` — Login / Sign-up (root entry)
- `dashboard.html` — main dashboard with exam cards
- `exam.html` — exam interface with question area and nav sidebar
- `results.html` — mock results summary
- `admin.html` — admin panel mock
- `css/` — styles (`main.css`, `exam.css`)
- `js/` — scripts (`auth.js`, `exam.js`, `main.js`)

Open `index.html` in a browser to preview. No build tools required.
